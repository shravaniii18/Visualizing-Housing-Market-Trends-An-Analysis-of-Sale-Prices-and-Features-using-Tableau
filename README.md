# Visualizing-Housing-Market-Trends-An-Analysis-of-Sale-Prices-and-Features-using-Tableau
Project Overview

This project analyzes housing market data to understand how different property features influence house sale prices. Using Tableau, the dataset was transformed into interactive visualizations and a dashboard to explore trends in housing characteristics such as renovation status, number of bedrooms, and house age.

The goal of the project is to help stakeholders gain insights into housing market behavior and make better data-driven decisions related to pricing strategies and property analysis.

Objectives

Analyze housing sales data using data visualization techniques.

Identify relationships between housing features and sale prices.

Understand the impact of renovations on property value.

Create an interactive dashboard for exploring housing market trends.

Tools and Technologies

Tableau Public – Data visualization and dashboard creation

CSV Dataset – Housing data used for analysis

GitHub – Project documentation and version control

Dataset Description

The dataset contains information about residential houses including:

Sale Price

Number of Bedrooms

Number of Bathrooms

Number of Floors

Basement Area (in Sqft)

House Age

Years Since Renovation

Renovation Status

These features were used to analyze trends in housing prices and property characteristics.

Data Visualizations
KPI Cards

The dashboard includes key metrics summarizing the dataset:

Total Houses

Average Sale Price

Total Basement Area

Renovation Distribution

A pie chart showing the proportion of houses that have been renovated versus those that have not.

Sales vs Renovation Years

A bar chart analyzing how the number of years since renovation affects housing prices.

Bedrooms vs Price

A visualization showing the relationship between the number of bedrooms and average house sale price.

House Age Distribution

A chart showing the distribution of houses based on their age.

Dashboard

All visualizations are combined into an interactive Tableau dashboard that allows users to explore housing trends and understand how different features influence property prices.

The dashboard provides a clear visual summary of the dataset and helps identify important patterns in the housing market.

Key Insights

Houses with more bedrooms generally have higher sale prices.

Renovated houses often show higher market value.

A large portion of houses in the dataset have not been renovated.

Housing price trends vary depending on structural features such as bedrooms and basement size.

Future Improvements

Include additional features such as location and property condition.

Implement predictive analysis for estimating house prices.

Enhance dashboard interactivity with additional filters and user controls.

Conclusion

This project demonstrates how Tableau can transform raw housing data into meaningful insights through visualization and interactive dashboards. The analysis helps understand housing market dynamics and supports data-driven decision making in real estate.
